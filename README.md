## Array
```
$angka = [1,2,3,4,5];
$buah = ["pia", "has", "ifa"];
```
Adalah bentuk array di php. bisa diakses dengan cara berikut:
```
echo $angka[1];
print_r($buah);
var_dump($buah);
```
```print_r``` akan menampilkan isi data array beserta indexnya. ```var_dump``` akan menampilkan isi data array beserta index dan tipe datanya.
